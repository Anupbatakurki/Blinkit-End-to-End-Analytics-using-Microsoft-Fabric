## Project Overview

This project demonstrates an end-to-end data analytics solution built using Microsoft Fabric. The solution follows the Medallion Architecture (Bronze → Silver → Gold), integrates machine learning for delivery delay prediction, and visualizes business insights through Power BI dashboards.

## Problem Statement

Blinkit processes thousands of customer orders every day. Business teams need a centralized analytics platform to monitor:

Revenue
Orders
Delivery performance
Store performance
Product performance
Late deliveries
Machine learning predictions

This project provides an end-to-end analytics pipeline to solve these challenges.
